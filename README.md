
# 👗 Astralys - Moda Futurista Gótica

Este es el sitio web de *Astralys*, una marca de ropa conceptual boliviana inspirada en una estética gótica futurista. El sitio fue desarrollado como parte de un proyecto universitario para la materia de Desarrollo Web.

## 💡 ¿Por qué este proyecto?

Quise crear una tienda que represente un estilo visual único y oscuro, pero con un enfoque moderno y funcional. Astralys es una propuesta que mezcla lo estético con lo tecnológico.

## 🚀 Tecnologías utilizadas

- HTML5, CSS3 y JavaScript
- Bootstrap para diseño responsivo
- GitHub Pages y Netlify para el despliegue
- GitHub Copilot y ChatGPT como asistencia de desarrollo

## 📋 Funcionalidades

- Página de inicio con presentación visual de la marca
- Página de productos con imágenes, precios y botón "Agregar al carrito"
- Sistema de carrito de compras dinámico
- Página de contacto con formulario validado
- Estilo responsivo para distintos dispositivos
